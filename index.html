About Me

Hello! I'm Michael Pignatelli, a student at the University of Pennsylvania studying computer science with a concentration in computer vision/AI.

I'm currently taking classes in computer systems, data structures, automata theory, probability, and machine learning for embedded systems.

I am currently a teaching assistant for CIS 1100: Intro to Computer Programming.

I'm passionate about developing technologies and their potential to drive positive change!

Projects

  • Pedestrian Buddies - Created as a final project for ESE 3600, this project involved the physical integration of a traffic signal with a computer vision system to detect gestures by a crossing guard. This project, done alongside Zeno Dancanet, involved the training of a model using a self-collected dataset of gesture images, taking advantage of the TensorFlow and TensorFlow Lite Python libraries.
  • Guess Who? - Created as a final project for CIS 1200, this is a Java Swing-based game inspired by the classic board game "Guess Who?".
